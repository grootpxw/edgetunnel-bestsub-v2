# 配置编辑功能说明

## ✨ 新增功能

在 EdgeTunnel BestSub v2 中，我们为"核心配置"卡片添加了**在线编辑和保存**功能，让你无需手动修改 YAML 文件即可调整常用配置。

---

## 🎯 可编辑的配置项

以下配置项支持在线编辑：

| 配置项 | 说明 | 验证规则 |
|--------|------|----------|
| **监听地址** | 服务器监听地址和端口 | 必填 |
| **Worker 地址** | Cloudflare Worker 的完整 URL | 必填 |
| **Worker 密码** | Worker 后台管理密码 | 必填 |
| **测速 URL** | 用于探测的目标 URL | 必填 |
| **保留数量** | 最终保留的最优 IP 数量 | 1-100 |
| **并发数** | 并发探测的数量 | 1-500 |
| **超时时间(ms)** | 单个 IP 的探测超时时间 | 100-30000 |

---

## 📖 使用方法

### 1. 进入编辑模式

点击"核心配置"卡片右上角的 **"编辑"** 按钮。

### 2. 修改配置

- 所有可编辑的配置项会变成输入框
- 密码字段在查看模式下显示为 `••••••••`
- 数字类型的配置会自动验证范围

### 3. 保存或取消

- 点击 **"保存"** 按钮：验证并保存配置到 `config.yaml`
- 点击 **"取消"** 按钮：放弃修改，恢复原始值

### 4. 保存成功

- 弹出成功提示
- 自动退出编辑模式
- 重新加载配置显示最新值

---

## ⚠️ 注意事项

### 1. **配置验证**
- 所有必填字段不能为空
- 数字类型必须在指定范围内
- 验证失败会弹出提示，不会保存

### 2. **保存机制**
- 配置保存到 `configs/config.yaml` 文件
- 保存后立即生效，无需重启程序
- 建议在修改前备份配置文件

### 3. **不可编辑的配置**
以下配置项仅显示，不支持在线编辑：
- 配置文件路径
- 端口列表
- 国家筛选（通过专门的国家选择器修改）
- Clash 目录
- Dry run 状态

### 4. **密码安全**
- 密码在查看模式下隐藏显示
- 编辑模式下使用 `type="password"` 输入框
- 保存时会完整保存到配置文件

## 🎨 界面特点

### 查看模式
- 配置以文本形式显示
- 密码显示为 `••••••••`
- 右上角显示"编辑"按钮

### 编辑模式
- 可编辑项变为输入框
- 输入框带有深色主题样式
- 聚焦时显示紫色边框和发光效果
- 右上角显示"保存"和"取消"按钮

---

## 🔧 技术实现

### 前端
- **HTML**: 添加编辑、保存、取消按钮
- **CSS**: 添加 `.config-input` 样式
- **JavaScript**: 
  - `enterEditMode()` - 进入编辑模式
  - `exitEditMode()` - 退出编辑模式
  - `saveConfigChanges()` - 保存配置
  - `renderConfigView()` - 根据模式渲染视图

### 后端
- **API**: `POST /api/config/update`
- **功能**: 接收配置更新请求，验证并保存到 YAML 文件
- **支持字段**:
  - `server.listen`
  - `worker.base_url`
  - `worker.password`
  - `probe.target.url`
  - `probe.keep`
  - `probe.concurrency`
  - `probe.timeout_ms`

## 📝 示例

### 修改监听端口

1. 点击"编辑"按钮
2. 将"监听地址"从 `127.0.0.1:8081` 改为 `0.0.0.0:8081`
3. 点击"保存"
4. 配置立即生效

### 调整并发数

1. 点击"编辑"按钮
2. 将"并发数"从 `100` 改为 `200`
3. 点击"保存"
4. 下次测速时使用新的并发数
